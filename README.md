# 🇮🇳 Discover India

> **A Journey Through Culture & Heritage** — An interactive, multi-page web project celebrating the richness of India's travel, heritage, and culture.

---

## 📖 About the Project

**Discover India** is a fully front-end web application built using pure **HTML5**, **CSS3**, and **Vanilla JavaScript**. It serves as a comprehensive travel guide and cultural platform for exploring India — from interactive state maps to trip planners, multimedia vaults, and a live impact dashboard.

The project was developed as part of an academic User Interface Design (UID) assignment by students of **Amrita School of Computing, Amritapuri**.

## 🚀 Features

| Page | Key Features |
|---|---|
| **Home** | Auto-playing hero slideshow, stats counter, upcoming events, travel news |
| **The Indian Canvas** | Google Charts interactive map, state-by-state search with autocomplete, photo gallery |
| **Top Destinations** | Ranked cards for India's top 10 travel spots with imagery and tags |
| **Plan Your Trip** | Preference-based itinerary generator, budget slider, destination autocomplete |
| **Traveller Essentials** | Visa info, weather guide, ethics & safety tips, video hero background |
| **Multimedia Vault** | Searchable image/video asset grid, featured audio player |
| **Impact Dashboard** | Animated counter metrics, four pillars of sustainable tourism |
| **My Profile** | Sign In / Sign Up form with localStorage persistence, saved itineraries |
| **Support & Connect** | Contact form, FAQ accordion, Google Maps embed, travel advisory |
| **Experiences** | Community review cards, interactive star-rating form, toast notification |

## 🛠️ Technologies Used

- **HTML5** — Semantic markup (`<header>`, `<main>`, `<article>`, `<aside>`, `<section>`, `<footer>`)
- **CSS3** — Custom properties, Flexbox, Grid, animations, glassmorphism
- **Vanilla JavaScript** — DOM manipulation, localStorage, form validation, intersection observers
- **Font Awesome 6.4** — Icon library (CDN)
- **Google Charts** — Interactive GeoChart for India map
- **External APIs** — Unsplash & Pexels images, LoremFlickr placeholders

## ▶️ How to Run

> No build tools or servers required. This is a pure static website.

1. **Clone or download** the project folder.
2. **Open `html/Home.html`** directly in any modern web browser (Chrome, Firefox, Edge).
3. Navigate between pages using the top navigation bar.

> ⚠️ Some features (like the Google GeoChart map) may require an active internet connection.
